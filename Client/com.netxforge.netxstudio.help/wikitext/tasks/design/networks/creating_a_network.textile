h1. Creating or editing a Network


!PLUGINS_ROOT/com.netxforge.netxstudio.help/img/tasks/design/networks/creating_a_network.png!


The following data should be entered or is presented.

*Info*

* The Network Name. The name of the Network.
* The Network Description. The description of the network. 

*Save* => Menu -> File -> Save
*Cancel* => The editing can be undone by Menu-> Edit -> Undo
