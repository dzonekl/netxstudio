h2. User roles

The following roles are defined: 

* Administrator 

* Planner

* Viewer


h3. Admininistrator

An administrator has full read and write access on all tasks of the system. Mainly the administrator as access to the Administrative task. 

h3. Planner

A planner has full read and write access to the tasks of the system, except the Administrator task.
