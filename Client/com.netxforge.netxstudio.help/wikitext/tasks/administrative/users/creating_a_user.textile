h1. Creating a user. 

When creating a User the following screen will appear.

!PLUGINS_ROOT/com.netxforge.netxstudio.help/img/tasks/administrative/users/creating_a_user.png!

The following data should be entered 

*General Information*

* The login
* The First name
* The Last name
* The Email address
* If the user is Active

*Authentication*

* The password, confirmed when entered twice. 

*Role*

* The role of the User.
