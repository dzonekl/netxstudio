h1. Tasks

The following Capacity Management tasks exist

* Administrative Task
* Library Task
* Monitoring Task
* Design Task

h2. Administrative task

The Administrative task provides access to administrative activities: 

* Users
* Jobs
* ...

h2. Library task

The Library task provides access to the activities concerning the library. 

* Expressions
* ...

h2. Monitoring task

The Monitoring task provides access to activities concerning Service and Resource monitoring

* Service Monitoring
* Resource Monitoring

h2. Design task

The Design task provides access to activities concerning the design of the network, network elements
and relationships between network elements.
