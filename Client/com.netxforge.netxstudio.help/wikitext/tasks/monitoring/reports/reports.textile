h2. Reports Information

There a two report types. 

* "Services Reports":PLUGINS_ROOT/com.netxforge.netxstudio.help/html/tasks/monitoring/services/services.html
* "Network Element Report":PLUGINS_ROOT/com.netxforge.netxstudio.help/html/tasks/design/resources/resources.html

h3. Service Report

A Service Report generates the following files: 

* Service Executive Summary Report
* Service Dashboard Report
* Service Resources Report
* Service User Report

h3. Network Element Report

A Network Element Report generates one file: 

* Network Element Resources Report
