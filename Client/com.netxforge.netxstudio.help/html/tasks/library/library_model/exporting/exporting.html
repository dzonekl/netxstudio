Exporting

The Library objects can be exported to various file formats.

Exporting formats.

  • Excel (.xls) format. => This format is the same as the import format for excel.

(Please refer to
importing for the correct format).

  • HTML (.html) format.

The layout and date in this export is fixed.

  • Portable Document format. (.pdf)

The layout and data is the same as html, but converted to .pdf.
