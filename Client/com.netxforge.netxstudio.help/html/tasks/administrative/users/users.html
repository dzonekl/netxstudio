Users

User management consists of viewing, creating and modifying users which can
access the system. Users have a single role, which determines the permission to use tasks of the user in the system.

User roles

The following roles are defined:

  • Administrator
  • Planner
  • Viewer

Admininistrator

An administrator has full read and write access on all tasks of the system. Mainly the administrator as access to the Administrative task.

Planner

A planner has full read and write access to the tasks of the system, except the Administrator task.

A viewer has has only read access to the tasks of the sytems, except the Administrator task.
