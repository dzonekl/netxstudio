Server Settings

Server settings provide access to several settings related to the Server side.

The Server Settings screen

Server Directories

Metric Sources Directories => The relative path on the server where metric source files can be found.

Note: The Metric Source location completes the full path from where files will be scanned and mapped.

Reports Directory => The absolute path to where the reports will be stored.

Note: Reports on the server can be browsed in the Monitoring Task.

Expansion Duration

Definition for the number of days, an expansion takes.

Note: The expansion duration is set on Functions and Equipments.
