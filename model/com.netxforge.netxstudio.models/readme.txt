Model plugin
============

Contains models in XSD, EMF, EEF, CDO, GMF etc.. 
The model revisions are maintained using a folder holding the date of the model update. 
The model date is also reflected in the Target namespace like this: 

1.0 http://www.netxforge.com/05012011/netxstudio
TMNL: http://www.netxforge.com/13042011/*
CallFlows/Tracing: http://www.netxforge.com/06032012
TMNL Addon: http://www.netxforge.com/16042013_delta/*
